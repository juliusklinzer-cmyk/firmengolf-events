## Pro-Artikel-Workflow

1. Claude schickt **Experten-Brief** (4–6 Fragen).
2. Julius antwortet in Stichpunkten/Sprachnotiz (Golf-/Praxiswissen, echte Beispiele/Zahlen).
3. Claude baut fertigen Post: Title ≤60 Z., 1× H1, Meta ≤160 Z., Keyword in ersten 100 Wörtern, interne Links, FAQ-Schema, passende Kategorie, Lesezeit/Autor.
4. Veröffentlichung + interne Verlinkung von bestehenden/neuen Posts.

## Konventionen
- Eine Seite, ein Haupt-Keyword (siehe `docs/seo-keyword-map.md`).
- Jeder Post verlinkt auf mind. 1 Format-Hub oder City-Page (Money-Page-Stützung).
- Echte Praxis/Erfahrung rein (E-E-A-T) — keine generischen KI-Floskeln.
