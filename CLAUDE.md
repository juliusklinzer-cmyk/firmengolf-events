## Project Overview

WordPress-based event marketplace and partner portal for Firmengolf (corporate golf events). The stack is a Docker Compose local dev environment; the only tracked custom code is the plugin and child theme.

## Local Development

```bash
# Start all services (WordPress, MariaDB, phpMyAdmin)
docker compose up -d

# Stop services
docker compose down

# View WordPress logs
docker compose logs -f wordpress

# Run WP-CLI commands inside the container
docker compose exec wordpress wp <command>
```

- WordPress: http://localhost:8080
- phpMyAdmin: http://localhost:8081

## Architecture

Only two directories are tracked in git (everything else in `wordpress/` is gitignored):

- `wordpress/wp-content/plugins/firmengolf-events/` — main plugin (`firmengolf-events.php`)
- `wordpress/wp-content/themes/firmengolf-child/` — child theme of Twenty Twenty-Five

The `db/` directory (MariaDB data volume) is also gitignored. `wordpress/wp-config.php` is gitignored — it is generated by WordPress on first run using the `WORDPRESS_DB_*` env vars in `docker-compose.yml`.
